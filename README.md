# aicli
Use ChatGPT in your terminal. Kind of feels like a bash utility?

## Installation
```bash
go get github.com/lunabrain-ai/aicli
```

## Usage
```bash
export OPENAI_API_KEY="<your openai api key>"
aicli "Write me a go function that prints 'Hello World'"
```

## Hack

### Run locally
```bash
go run main.go
```

### Build
```bash
go build main.go
```
